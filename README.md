### Команды
Переход директорию app:
```
    cd app
```

Виртуальное окружение:
```
    python -m venv venv
```
Активация:
```
    .\venv\Scripts\activate
```
Установка библиотек:
```
pip install -r requirements.txt
```
Запуск бота:
```
    python bot.py
```
